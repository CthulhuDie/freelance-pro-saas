# 🚀 Freelance Pro: Ghostwriter Platform MVP

Este proyecto es un **SaaS (Software as a Service)** diseñado para freelancers que buscan automatizar su negocio con Inteligencia Artificial. Es una plataforma de aprendizaje y herramientas financieras con un flujo de usuario completo.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5 semántico y CSS3 con variables dinámicas.
* **Lógica:** JavaScript ES6+ (Manipulación del DOM y arquitectura SPA).
* **Pagos:** Integración oficial con el **SDK de PayPal**.
* **IA Concept:** Flujo de Ghostwriting para generación de contenido profesional.

## ✨ Funcionalidades Clave
1. **Sistema de Checkout Real:** Integración con botones inteligentes de PayPal para desbloqueo de contenido.
2. **Calculadora Freelance:** Herramienta lógica que calcula la tarifa por hora basada en gastos y margen de beneficio.
3. **Navegación SPA:** Cambio de secciones sin recarga de página para una experiencia de usuario fluida.
4. **Manual Maestro (PDF):** Sistema de entrega de recursos premium mediante ventanas modales.
